#
# Overview

```ts twoslash
import { htmlToMarkdown, htmlToMdast } from "webforai";
import { loadHtml } from "webforai/loaders/playwright";

// Load html from url
const url = "https://www.npmjs.com/package/webforai";
const html = await loadHtml(url); // [!code hl]

// Convert html to markdown
const markdown = htmlToMarkdown(html, { baseUrl: url }); // [!code hl]
```

#
# Features

- High-quality HTML to Markdown conversion with simple, customizable options
- ESM-native, compatible with various environments (browser, Cloudflare Worker, Node.js, etc.)
- Lightweight (only 146kb minified and gzipped)
- Flexibility because it's built on [Syntax Tree](https://github.com/syntax-tree) ecosystem.
- MathML to LaTeX with [mathml-to-latext](https://www.npmjs.com/package/mathml-to-latex) package.
- Out-of-the-box loaders such as fetch, [playwright](https://www.npmjs.com/package/playwright), [puppeteer](https://www.npmjs.com/package/puppeteer).
- CLI tool `npx webforai` for quick HTML to Markdown conversion
